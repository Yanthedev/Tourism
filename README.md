🌍 Landing Page de Turismo

✨ Sobre o Projeto

Este projeto consiste no desenvolvimento de uma Landing Page de turismo, construída utilizando apenas HTML e CSS.

A página foi criada com o objetivo de apresentar informações relevantes sobre destinos turísticos, destacando:

📍 Locais atrativos
🖼️ Imagens envolventes
📝 Textos descritivos

Tudo isso organizado de forma clara, moderna e visualmente agradável.

🎯 Objetivo

O principal foco deste projeto é o aprimoramento das seguintes habilidades:

- Estruturação semântica com HTML
- Estilização e layout com CSS
- Organização visual de conteúdo
- Construção de interfaces limpas e responsivas

🛠️ Tecnologias Utilizadas
- HTML5
- CSS3

🚀 Sobre o Desafio

Este projeto faz parte de um dos desafios propostos pela Rocketseat, dentro da jornada de aprendizado FullStack. Ele representa mais um passo na construção de uma base sólida em desenvolvimento web.
